# .NET LORQ implementation

This directory is reserved for the final LORQ v1 implementation.

Planned projects:

- `Lorq.Cli` - command-line entry point.
- `Lorq.Core` - experiment, run shard, merge, package, judgement, and report domain model.
- `Lorq.Adapters.Copilot` - first-class industrial Copilot SDK adapter.
- `Lorq.Adapters.Process` - one-shot file-based external adapter protocol and Codex process adapter foundation.
- `Lorq.Reporting` - JSON-canonical report model and Markdown renderer.

The .NET implementation should start only after the Python v0 deterministic orchestration baseline is frozen.
