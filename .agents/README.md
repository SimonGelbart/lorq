# LORQ agent rules

This directory contains reusable operating rules for AI agents working on the LORQ repository.

These rules are source-controlled because they preserve product direction and contribution hygiene across sessions. They are not scratch notes, generated logs, or private handoff material.

Start with:

- `LORQ_AGENT_RULES.md` for mandatory repo, roadmap, validation, and handoff rules.

Session-specific notes, generated outputs, local scripts, logs, extracted archives, and one-off handoffs must stay outside this repository under the sibling `internal/` workspace directory used by delivery artifacts.
